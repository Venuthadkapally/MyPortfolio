PROJECT USING DOUBLY LINKED LIST

AirLine Online Ticket Booking

  • About project?

    • This project is an online airline ticket booking system that allows users to search, compare, and book flight tickets efficiently. It ensures a smooth ticket management process by using a Doubly Linked List (DLL) for handling bookings.

  • How did I do it?

    • Implemented Doubly Linked List (DLL) to store and manage booking records, allowing efficient insertion, deletion, and modification of tickets. The system enables easy navigation between booked tickets (both forward and backward), ensuring a well-structured database for seat availability, cancellations, and rebookings.

  • Impact / Why is this helpful?

    • Using a Doubly Linked List improves the efficiency of managing flight bookings, cancellations, and modifications. It enhances search speed, reduces processing time, and ensures smooth data handling, making the booking system more reliable and user-friendly.

IOT BASED PROJECT USING AURDINO

Health Monitoring System

  • About project?

    • The project is about building an IoT-based health monitoring system that can track a patient’s vital signs—like body temperature, oxygen (SpO₂) level, and heart rate—in real time. The data is sent to a mobile app using Bluetooth, enabling remote health tracking.

  • How did I do it?

    • We used sensors (MAX30100 and LM35), a microcontroller (ESP8266), and the MIT App Inventor to create a mobile app. The system reads data from the body, processes it using the Arduino board, and displays it on an LCD screen and the app. The setup is tested for accuracy and real-time performance.

  • Impact / Why is this helpful?

    • This system helps patients, especially in rural areas, monitor their health from home without needing frequent hospital visits. It supports early detection of issues like low oxygen or high fever, and allows doctors to remotely monitor patient data, making healthcare more accessible and affordable.

PROJECT USING MACHINE LEARNING

Cyber Crime Type & Occurance Prediction Using Machine Learning

  • About project?

    • This project focuses on predicting crimes using Machine Learning techniques. It analyzes historical crime data to identify patterns, classify crime types, and predict high-risk locations, helping law enforcement take preventive measures.

  • How did I do it?

    • Collected and preprocessed crime data, performed Exploratory Data Analysis (EDA) to identify trends, and implemented ML models like Naïve Bayes, SVM, Decision Trees, Random Forest, and LSTM. Additionally, used geospatial mapping to visualize high-crime areas and evaluated model performance for accuracy.

  • Impact / Why is this helpful?

    • The system assists law enforcement in predicting and preventing crimes, improving resource allocation, and ensuring public safety. By providing real-time crime forecasts, it enables smarter decision-making and helps reduce crime rates.
